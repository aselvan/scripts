# Browser Extensions

### Collection of random browser extensions

- #### /linkedin-extension-blocker
  This is a simple extension that injects an interceptor code to catch LinkedIn’s attempts to load extension resources and deny those requests. Other than that, the extension does absolutely nothing else.

- #### /extension-telemetry-demo
  This is a simple extension to demonstrate how much data a browser extension with right permission can spy on your browsing activity.
